    EbanqApi¶ ↑

    A wrapper for the EBANQ Rest API. Specification is as described in the the developer documentation. This gem is a work in progress.

    Installation¶ ↑

    Add this line to your application's Gemfile:

    gem 'ebanq_api'

    And then execute:

    $ bundle

    Or install it yourself as:

    $ gem install ebanq_api

    Usage¶ ↑

    Before you start making the requests to EBANQ API provide the base_url using the configuration wrapping. ruby EbanqApi.config do |config| config.base_url = 'http://yourebanqdomain.com' config.token = 'put your user token here' config.secret = 'put your user secret here' end 

    client = EbanqApi::Client.new

client.profile.show(1)
client.profile.update(1, {first_name: 'John', last_name: 'Doe'})
client.auth.request_new_password('John')
