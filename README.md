# EbanqApi

A Ruby wrapper for the EBANQ Rest API.
This gem is a work in progress. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ebanq_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ebanq_api
